Powerful Features

Everything you need for professional tank monitoring

Real-time Monitoring

Live display of tank levels and temperatures with automatic updates every 3 seconds.

CSV File Import

Flexible file import with auto-discovery and mapping of CSV data to tank values.

Serial Communication

Direct serial port communication with configurable parameters and protocols.

Smart Alarms

Configurable pre-alarm (86%) and overfill (97.5%) thresholds with visual indicators.

Multiple View Modes

Switch between grid, list, and column layouts with drag & drop tank reordering.

Cross-platform

Native desktop applications for Windows, macOS, and Linux with Electron.
